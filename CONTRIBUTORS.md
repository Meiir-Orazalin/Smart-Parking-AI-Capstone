## Meiir Orazalin's portfolio contribution

The `mobile-app/` directory contains the Expo/React Native client maintained in Meiir's capstone workspace. It includes driver and operations flows, API integration contracts, mock/hybrid data handling, camera and alert workflows, and an interactive parking-lot busyness view.

Because the downloaded snapshot did not contain Git metadata, the original commit history is not reproduced in this mirror.
